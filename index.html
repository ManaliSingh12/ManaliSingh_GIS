Choropleth Map: Global Water Security

To visualize global progress toward SDG Target 6.1 using data from 2024, highlighting the massive disparities between countries with 100% safe water access regions with critical infrastructure or data gaps. This project maps the proportion of national populations with access to safely managed drinking water services. It uses a six-tier color gradient to highlight regional trends, distinctively marking nations with full 100% access in green and indicating missing data in grey.

Vector Data and Analysis

Spatial analysis of walking access from residences to physiotherapist clinics in Utrecht, Netherlands was conducted using QGIS to generate network-routing isochrone layers and identify peripheral service gaps. This was combined with a Volunteered Geographic Information (VGI) density map exploring data distribution by gender.

Raster elevation models

This analysis uses distinct green and indigo-to-cyan color ramps to visually separate modern surface objects (DSM) from bare-earth terrain (DTM), using a 40% transparency overlay to blend a 1927 historic map sheet with modern data. The spatial patterns reveal Wageningen’s modern urban grid and tree canopies contrasting sharply against raw underlying features like ancient paleochannels and the steep ice-pushed ridge boundary. Mapping these low-lying elevation gradients provides critical insights for sustainability and climate resilience, offering essential data for local flood mitigation initiatives like the Dutch Room for the River program and urban heat island management.

Vegetation Indices and Environmental changes

This analysis leverages multitemporal satellite imagery within Google Earth Engine to compute the Normalized Difference Vegetation Index (NDVI) across the Finnish Lakeland region. By comparing true-color composites against surface reflectance datasets, this project tracks seasonal canopy density, forest health trends and changes in dense vegetation distribution over time, demonstrating advanced workflows in cloud-based remote sensing.

Land use classification

Utilizing supervised pixel-based classification algorithms in QGIS, this project categorizes regional landscapes into distinct land cover classes including urban zones, agricultural parcels, forest cover and water bodies.

Multicriteria Analysis

This project applies Multi-Criteria Evaluation (MCE) techniques to conduct a comprehensive spatial vulnerability and seismic risk assessment for Vila Franca do Campo on São Miguel Island, Portugal. By standardizing and weighting physical, environmental and socio-economic constraint layers through map algebra, the final model successfully delineates high-risk mitigation zones to aid in local disaster resilience and sustainable evacuation planning.
